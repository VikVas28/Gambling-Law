# Чекори и промптови за Claude Code

Овој фајл ти кажува точно што да правиш за да ја изградиш апликацијата со **Claude Code**. Држи го `CLAUDE.md` во истата папка — Claude Code го чита автоматски.

---

## Чекор 1 — Подготовка
1. Инсталирај Node.js 18+ (со него доаѓа `npm`).
2. Инсталирај Claude Code:
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```
3. Направи папка за проектот и стави го `CLAUDE.md` во неа:
   ```bash
   mkdir gambling-law-map && cd gambling-law-map
   # копирај го CLAUDE.md тука
   claude
   ```

---

## Чекор 2 — Стартен промпт (копирај го цел во Claude Code)

```
Прочитај го CLAUDE.md во коренот. Изгради ја апликацијата опишана таму.

Започни со MVP (секцијата „Функционалности → MVP“):
- Vite + React + TypeScript + Tailwind + Leaflet/react-leaflet + @turf/turf.
- Постави ја структурата на проектот од CLAUDE.md §5.
- Имплементирај ја логиката за класификација точно по §2 (правилото за 500 m,
  и разликата помеѓу casino/automat_club/electronic_games наспроти betting_shop).
- Мапата нека е центрирана на Скопје, со 500 m кругови околу секое училиште
  и маркери обоени по статус (зелено/килибар/црвено).
- Направи мали тест-податоци (3–4 училишта и 4–5 објекти, verified:false) во
  public/data/ за да се види логиката како работи веднаш.
- UI на македонски, респонзивно за мобилен, со легенда, бројачи, филтри и
  страница „За законот“ со одредбите и изворите од §7 и §10.
- Стави видлив дисклејмер и ознака „непроверена локација“ за verified:false.

Пред да пишуваш код, накратко изложи план и потврди ја структурата на фајлови.
```

## Чекор 3 — Промптови за продолжување (по фази)

**Реални училишта од OSM:**
```
Додај модул src/lib/overpass.ts што ги повлекува училиштата од Overpass
(барањето за amenity=school од CLAUDE.md §7), нормализира во School[] и
има копче „Вчитај училишта од OpenStreetMap“ во панелот, со graceful fallback
ако мрежата е блокирана.
```

**Реални објекти (геокодирање на регистарот):**
```
Направи помошна скрипта scripts/geocode.ts што чита листа со адреси од
регистарот на Министерството за финансии (CSV/JSON) и ги геокодира преку
Nominatim во lat/lng, со паузи за rate-limit, и снима во public/data/venues.json
со source:"Министерство за финансии", verified:false.
```

**Режим за уредување + извоз:**
```
Имплементирај го EditPanel (CLAUDE.md §6 V2): додавање/поместување/бришење
точки на мапа, уредување на полиња, и копчиња за извоз/увоз на JSON.
Без localStorage — состојбата во меморија + извоз во фајл.
```

**Деплој:**
```
Подготви за деплој на GitHub Pages (или Vercel): build скрипта, base path,
и кратко упатство во README.
```

## Чекор 4 — Внеси ги реалните податоци
- **Училишта:** изврши го Overpass барањето (CLAUDE.md §7) на https://overpass-turbo.eu, извези GeoJSON/JSON, претвори во `schools.json`.
- **Објекти:** отвори ја листата лиценци на Министерството за финансии
  (`https://finance.gov.mk/mk-MK/oblasti/licenci-za-igri-na-srekja`), извлечи ги адресите и приредувачите, геокодирај ги, и **рачно провери** ги локациите пред да ставиш `verified: true`.

## Совет
Работи во мали чекори и комитирај често (`git`). По секоја фаза кажи му на Claude Code што да направи следно — едно по едно. Ако нешто не изгледа добро, опиши го конкретно (пр. „кругот од 500 m е преголем“ или „на мобилен листата го покрива копчето“) и побарај поправка.
