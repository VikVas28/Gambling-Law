# CLAUDE.md — Мапа за усогласеност со Законот за игри на среќа (СРМ, 2026)

> Овој фајл е контекст за **Claude Code**. Стави го во коренот на репозиториумот — Claude Code го чита автоматски. Истовремено служи и како читлива спецификација за човек.

---

## 1. Што градиме

Веб-апликација (мапа) што визуелно покажува **кои објекти за игри на среќа мора да се релоцираат, кои може да останат со ограничување, и кои се сообразни**, според новиот Закон за игри на среќа усвоен во Собранието на СРМ во 2026 (59 гласа „За“, 9 „Против“).

Клучното правило: казината, автомат клубовите и приредувачите на електронски игри на среќа мора да бидат **најмалку 500 метри (радиус) од сите основни и средни училишта**. Апликацијата го пресметува тоа растојание автоматски и го бои секој објект според статусот.

**Целна публика:** граѓани, новинари, локална самоуправа, инспекција. **Една главна задача на страницата:** на мапа да се види усогласеноста со правилото за 500 метри.

## 2. Точни правила од законот (логика за класификација)

Растојанието е **праволиниско (радиус)**, не пешачко — законот користи формулација „радиус до 500 метри“.

| Тип на објект (`kind`) | Во радиус ≤ 500 m од училиште | Надвор од 500 m |
|---|---|---|
| `casino` (казино) | **МОРА ДА СЕ РЕЛОЦИРА** | Сообразен |
| `automat_club` (автомат клуб) | **МОРА ДА СЕ РЕЛОЦИРА** | Сообразен |
| `electronic_games` (приредувач на е-игри на среќа) | **МОРА ДА СЕ РЕЛОЦИРА** | Сообразен |
| `betting_shop` (обложувалница / уплатно-исплатно место) | **ОСТАНУВА СО ОГРАНИЧУВАЊЕ** — забранети се видеолотариски терминали (VLT) и опрема за е-игри во радиусот | Сообразен |

Три статуси во кодот:
- `must_relocate` — црвено
- `restricted` — портокалово/килибар
- `compliant` — зелено

Псевдокод:
```ts
function classify(venue: Venue, schools: School[]): Status {
  const minDist = Math.min(...schools.map(s => distanceMeters(venue, s)));
  if (minDist > 500) return "compliant";
  if (venue.kind === "betting_shop") return "restricted";
  return "must_relocate";
}
```
Растојанието: користи `@turf/distance` (units: meters) или Haversine. Радиусот е спрема **најблиското** училиште; ако има барем едно училиште во 500 m → неусогласен/со ограничување.

**Останати одредби од законот** (НЕ влегуваат во просторната логика, но прикажи ги на страницата „За законот“):
- Забрана за рекламирање на излозите на коцкарниците.
- Забрана за реклами што коцкањето го прикажуваат како пат до личен успех / брза заработка.
- Забрана за рекламирање преку познати личности.
- Задолжителен GPS на секој апарат за тековна контрола од Управата за јавни приходи (УЈП).
- Лиценците не може да се мултиплицираат.
- Финансиски давачки за нови уплатно-исплатни места и повисоки банкарски гаранции.
- Имплементација на препораките на MONEYVAL (Совет на Европа).

## 3. Tech stack

- **Vite + React + TypeScript** — статична SPA, без бекенд за MVP.
- **Tailwind CSS** — стилизирање.
- **Leaflet + react-leaflet** — мапа (едноставно, стабилно). Алтернатива: MapLibre GL за векторски подлоги.
- **@turf/turf** — геодезиски пресметки (растојание, опционо буфери).
- **leaflet.markercluster** — групирање маркери кога има многу точки.
- **Zustand** (опционо) или само React state — состојба на апликацијата.
- Подлога (basemap): CARTO Positron или OSM raster tiles.
- Хостинг: **Vercel / Netlify / GitHub Pages** (целосно клиентски, статичен сајт).

Зошто без бекенд: податоците се мал број статични JSON фајлови во репото; ажурирањето е преку git. Ако подоцна треба админ-уредување во живо, додај лесен бекенд (Supabase/PocketBase) — види §8.

## 4. Модел на податоци

```ts
// src/lib/types.ts
export type SchoolType = "primary" | "secondary";
export type VenueKind = "casino" | "automat_club" | "electronic_games" | "betting_shop";
export type Status = "compliant" | "restricted" | "must_relocate";

export interface School {
  id: string;
  name: string;
  type: SchoolType;
  lat: number;
  lng: number;
  municipality?: string;
  address?: string;
  source: string;        // пр. "OSM", "МОН"
}

export interface Venue {
  id: string;
  name: string;
  kind: VenueKind;
  operator?: string;     // друштво/приредувач
  licenseNo?: string;    // број на лиценца
  lat: number;
  lng: number;
  municipality?: string;
  address?: string;
  source: string;        // пр. "Министерство за финансии", "OSM"
  verified: boolean;     // дали локацијата е рачно проверена
  // пресметани во рантајм, не се чуваат:
  // status, nearestSchoolId, nearestSchoolDistanceM
}
```

Податоците стојат во `public/data/schools.json` и `public/data/venues.json`. Статусот **не** се чува во JSON — се пресметува секогаш во рантајм од актуелните училишта.

## 6. Функционалности

**MVP (прв спринт):**
1. Мапа на С. Македонија, центрирана на Скопје (`[41.9981, 21.4254]`, zoom ~12).
2. Училишта како маркери + полупроѕирен **круг од 500 m** (зона на ограничување) околу секое.
3. Маркери за објекти, обоени по статус (зелено/килибар/црвено), со икона по тип.
4. Автоматска класификација според §2.
5. Легенда + бројачи: колку „мора да се релоцираат“, „со ограничување“, „сообразни“.
6. Попап/детали при клик (име, тип, статус, најблиско училиште и растојание, извор).
7. Филтри и пребарување (по тип, статус, општина, име).
8. Респонзивно — на мобилен, листата/панелот како долен лист (bottom sheet).

**V2:**
- Режим за уредување: додај/помести/избриши точка, па **извоз во JSON** (download) и увоз.
- Повлекување училишта и објекти во живо од **OSM (Overpass)** — види §7.
- Геокодирање на адреси од регистарот преку **Nominatim**.
- Филтер по општина, споделлив линк со состојба (URL params), „измери растојание“.
- Кластерирање маркери (leaflet.markercluster) за перформанси.
- Повеќејазичност: mk (примарно), опционо sq и en.

## 7. Извори на податоци (КЛУЧНО)

### Училишта
- **OpenStreetMap / Overpass API** (`amenity=school`). Готово барање:
```overpassql
[out:json][timeout:60];
area["ISO3166-1"="MK"][admin_level=2]->.mk;
(
  node["amenity"="school"](area.mk);
  way["amenity"="school"](area.mk);
  relation["amenity"="school"](area.mk);
);
out center;
```
- Вкрстено провери со официјалната листа основни/средни училишта од МОН/општините.

### Објекти за игри на среќа
- **Министерство за финансии — Лиценци за приредување игри на среќа:**
  `https://finance.gov.mk/mk-MK/oblasti/licenci-za-igri-na-srekja`
  Ова е авторитетниот регистар. Секоја лиценца ја содржи **точната адреса на деловната просторија** (пр. „ул. Маршал Тито бр. 108, Богданци“) и приредувачот. Овие адреси се **геокодираат** (Nominatim) во координати.
- **Управа за јавни приходи (УЈП)** — надлежна за надзор и техничко-просторни услови: `http://ujp.gov.mk`.
- **OSM (Overpass)** како почетна/проверочна листа:
```overpassql
[out:json][timeout:60];
area["ISO3166-1"="MK"][admin_level=2]->.mk;
(
  nwr["amenity"="casino"](area.mk);
  nwr["leisure"="adult_gaming_centre"](area.mk);
  nwr["shop"="bookmaker"](area.mk);
  nwr["amenity"="gambling"](area.mk);
);
out center;
```

Препорачан тек: повлечи од МОН + Финансии → геокодирај адреси → рачна проверка (`verified: true`) → сними во `venues.json`/`schools.json`.

### Пример JSON

## 8. Опционен бекенд (само ако треба уредување во живо)
- **Supabase** (Postgres + PostGIS): чувај точки, прави просторни прашања (`ST_DWithin(..., 500)`) на серверска страна, додај админ-автентикација. Инаку MVP останува целосно статичен.

## 9. Дизајн насоки
- Граѓански „dashboard“ тон: чисто, институционално, без украси. Светла мапа, **темен страничен панел** (slate/navy).
- Статусни бои: зелена `#16a34a` (сообразен), килибар `#d97706` (со ограничување), црвена `#dc2626` (релокација); училишта индиго `#4f46e5`; зона од 500 m — полупроѕирен црвеникав круг.
- Икони по тип објект (казино/автомат/обложувалница) + статусна боја на маркерот.
- Целиот UI текст на **македонски**. Сис-фонт стек со кирилична поддршка.
- Потпис на страницата: преклопувачките зони на 500 m како главен визуелен момент + јасна легенда.
- Под „quality floor“: респонзивно до мобилен, видлив фокус за тастатура, `prefers-reduced-motion`.

## 10. Правни и етички предупредувања (мора да се почитуваат)
- Прикажувањето дека именуван бизнис „мора да се затвори“ има реални последици. **Не објавувај неверификувани локации како факт.** За `verified: false` стави видлива ознака „непроверена локација“.
- Стави дисклејмер: ова е **информативна алатка, не официјален документ**; меродавни се регистрите на Министерството за финансии и УЈП.
- Цитирај извор за секоја точка (`source`).
- Растојанието е праволиниско (радиус) според формулацијата во законот; ако официјалниот подзаконски акт пропише друг метод на мерење, ажурирај ја логиката.

## 12. Дефиниција на „готово“ за MVP
- Се вчитуваат `schools.json` и `venues.json`; сите објекти се класифицирани и обоени точно според §2.
- 500 m зоните се видливи и точни; бројачите се совпаѓаат со листата.
- Филтри/пребарување работат; мобилен приказ е употреблив.
- Страница „За законот“ со одредбите и изворите (Министерство за финансии, УЈП, Собрание).
- Дисклејмер и ознаки за непроверени локации присутни.
